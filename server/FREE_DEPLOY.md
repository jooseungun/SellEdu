# 무료 백엔드 서버 배포 가이드

## 💰 비용: 완전 무료!

Railway와 Render는 **무료 티어**를 제공하므로 초기 비용 없이 백엔드 서버를 배포할 수 있습니다.

## 🚀 Railway 무료 티어

### 제공 사항
- **월 $5 크레딧** 무료 제공
- 소규모 프로젝트에 충분한 리소스
- MySQL 데이터베이스 포함
- 자동 HTTPS
- GitHub 연동 자동 배포

### 제한 사항
- 월 $5 크레딧 초과 시 유료 플랜 필요
- 소규모 프로젝트/개발용으로 적합
- 트래픽이 많으면 크레딧 소모가 빠름

### 예상 사용량
- 소규모 프로젝트: 월 $5로 충분
- 중간 규모: 월 $10-20 정도
- 대규모: 유료 플랜 필요

## 🚀 Render 무료 티어

### 제공 사항
- **무료 웹 서비스** 배포 가능
- PostgreSQL 데이터베이스 무료 제공
- 자동 HTTPS
- GitHub 연동 자동 배포

### 제한 사항
- **15분 비활성 시 슬리프 모드** (첫 요청 시 깨어남)
- CPU/메모리 제한
- 소규모 프로젝트에 적합

### 예상 사용량
- 개발/테스트용: 완전 무료
- 소규모 프로덕션: 무료 티어로 가능
- 중간 규모 이상: 유료 플랜 필요

## 📊 비교표

| 기능 | Railway | Render |
|------|---------|--------|
| 무료 크레딧 | 월 $5 | 무제한 |
| 데이터베이스 | MySQL (포함) | PostgreSQL (무료) |
| 슬리프 모드 | 없음 | 15분 비활성 시 |
| 자동 배포 | ✅ | ✅ |
| HTTPS | ✅ | ✅ |
| 추천 용도 | 개발/소규모 | 개발/테스트 |

## 🎯 추천 배포 방법

### 개발/테스트용
**Render 추천**
- 완전 무료
- 슬리프 모드 있어도 개발용으로 충분
- PostgreSQL 무료 제공

### 소규모 프로덕션
**Railway 추천**
- 월 $5 크레딧으로 소규모 운영 가능
- 슬리프 모드 없음
- 더 안정적인 성능

## 💡 비용 절감 팁

### 1. 개발 환경 최적화
- 불필요한 로그 줄이기
- 데이터베이스 쿼리 최적화
- 캐싱 활용

### 2. 모니터링
- Railway/Render 대시보드에서 사용량 확인
- 크레딧 소모 추적

### 3. 필요 시 업그레이드
- 트래픽 증가 시에만 유료 플랜 고려
- 무료 티어로 시작 후 필요 시 확장

## ⚠️ 주의사항

1. **무료 티어 제한**
   - 리소스 제한이 있으므로 대규모 트래픽에는 부적합
   - 프로덕션 대규모 서비스는 유료 플랜 필요

2. **정책 변경**
   - 무료 티어 정책은 변경될 수 있음
   - 최신 정보는 공식 사이트 확인

3. **데이터 백업**
   - 무료 티어도 데이터 백업은 중요
   - 정기적으로 백업 수행
